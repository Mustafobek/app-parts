## Description

[App-parts](https://github.com/mustafobek/app-parts/src/shared/services) contributes reusable services for apps.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
